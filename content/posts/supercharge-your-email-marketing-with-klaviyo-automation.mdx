## Building Automated Workflows

Klaviyo's automation features allow you to create complex workflows that are triggered by specific events or actions. For example, you can create a workflow that sends a welcome email to new subscribers, or a workflow that sends a abandoned cart reminder to customers who have left items in their cart. To build a workflow, you'll need to define the trigger event, the actions that will be taken, and the conditions that must be met.

```javascript
// Example of a Klaviyo workflow trigger
const trigger = {
  type: 'subscription',
  event: 'newSubscriber',
  filters: [
    {
      property: 'email',
      operator: 'contains',
      value: '@example.com'
    }
  ]
};
```

<Callout type="tip">
Use Klaviyo's visual workflow editor to build and manage your automated workflows. This editor provides a intuitive interface for creating complex workflows without needing to write code.
</Callout>

## Personalizing Your Campaigns

Klaviyo's personalization features allow you to create targeted campaigns that are tailored to individual subscribers. You can use data such as subscriber preferences, purchase history, and behavior to create highly personalized emails that drive engagement and conversions. For example, you can create a campaign that sends a special offer to subscribers who have purchased a specific product in the past.

```javascript
// Example of a Klaviyo campaign personalization
const campaign = {
  name: 'Summer Sale',
  triggers: [
    {
      type: 'purchase',
      product: 'summer-product'
    }
  ],
  actions: [
    {
      type: 'sendEmail',
      email: 'summer-sale-email'
    }
  ]
};
```
